## 🚀 Projetos em Destaque & Experiência Técnica

### 📦 [Motoboy Express](https://github.com/ANTONIOALGMAR) - *Ecossistema de Logística Industrial*
Este é o meu projeto de maior complexidade técnica, onde apliquei conceitos avançados de engenharia de software:
- **Desafio de Tempo Real:** Implementei WebSockets para garantir que o administrador veja a posição exata do motoboy no mapa sem necessidade de refresh.
- **Arquitetura Monorepo:** Organizei o Frontend (Next.js), Mobile (React Native) e Backend (Fastify) de forma integrada, facilitando a manutenção e o compartilhamento de tipos (TypeScript).
- **Performance:** Utilizei **Redis** para cache de geolocalização, reduzindo drasticamente a carga no banco de dados principal.
- **UX para o Entregador:** O app mobile foi desenhado para ser operado com mínima fricção, essencial para quem está no trânsito.

### ☕ [DadosPrimitivos](https://github.com/ANTONIOALGMAR) - *Fundamentos de Engenharia*
Um projeto dedicado ao domínio do **Java**. Acredito que para ser um bom desenvolvedor senior, é preciso dominar a base:
- Estudo profundo de tipos de dados, alocação de memória e lógica estruturada.
- Exercícios de algoritmos para otimização de processamento.
